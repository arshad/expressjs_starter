Expressjs-starter
=================

Template for Express app with:

* Backbonejs
* Requirejs
* Jade
* Bower Package Manager

### Installation

Install node dependencies

    cd /path/to/app
    npm install

Install components with Bower

    npm install bower -g
    cd public
    bower install --save

Run the app

    cd /path/to/app
    node server.js
